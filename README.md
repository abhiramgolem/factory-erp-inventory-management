# Factory ERP Inventory Management System

A custom ERP system for plastic manufacturing factories.

## Features

- In Load Management
- Out Load Management
- Inventory Tracking
- Production Management
- Waste Sales
- Employee Attendance
- Salary Calculation
- Expense Tracking
- Profit Calculation
- Reports and Analytics

## Tech Stack

- Frontend: React
- Backend: Node.js + Express
- Database: PostgreSQL
- Version Control: Git + GitHub
